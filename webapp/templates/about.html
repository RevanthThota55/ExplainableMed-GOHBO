About ExplainableMed-GOHBO

Project Overview

ExplainableMed-GOHBO is an advanced medical image classification system that combines state-of-the-art deep learning with hybrid meta-heuristic optimization and explainable AI techniques.

The system uses a ResNet-18 architecture optimized with the GOHBO algorithm (Grey Wolf + Orthogonal Learning enhanced Heap-Based Optimization) to achieve superior performance in brain tumor classification from MRI scans.

GOHBO Algorithm

The GOHBO (Grey Wolf + Orthogonal Learning enhanced Heap-Based Optimization) algorithm is a hybrid meta-heuristic that combines three powerful optimization techniques:

Grey Wolf Optimizer (GWO)

Mimics wolf pack hunting behavior with alpha, beta, and delta wolves guiding the search for optimal hyperparameters.

Heap-Based Optimizer (HBO)

Uses efficient heap data structures to maintain and quickly access the best solutions during optimization.

Orthogonal Learning (OL)

Enhances solution diversity using orthogonal experimental design to prevent premature convergence.
